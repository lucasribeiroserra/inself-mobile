# Assets

Estrutura alinhada ao Inself:

## Raiz (ícones do app)

- `icon.png` — ícone do app (1024x1024)
- `splash-icon.png` — imagem da tela de splash
- `adaptive-icon.png` — ícone adaptativo Android (1024x1024)
- `favicon.png` — favicon para web

## images/

- `logo.png` — logo principal (header, auth)
- `placeholder.png` — imagem padrão para cards/conteúdo vazio

Para gerar ícones a partir de um único PNG: https://www.appicon.co/ ou `npx expo prebuild` após configurar.
